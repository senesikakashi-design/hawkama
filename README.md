## 🎯 كيفية الاستخدام

### إدارة الصلاحيات:
1. سجل دخول كـ compliance
2. اذهب لـ "إدارة الصلاحيات"
3. اختر مستخدم
4. حدد الصلاحيات
5. احفظ

### الإشعارات:
- تظهر تلقائياً عند إنشاء طلب جديد
- اضغط على الجرس لعرض القائمة
- اضغط على الإشعار للذهاب للطلب

### الباك آب:
1. سجل دخول كـ compliance
2. اذهب لـ "الإعدادات"
3. اضغط "تحميل نسخة احتياطية"
4. احفظ الملف في مكان آمن
